# Evolution Game Documentation

## Overview
The Evolution Game is a simulation that showcases the principles of evolution and natural selection. Players can create and evolve organisms to see how they adapt to different environments.

## Features
- **Create Organisms**: Players can design organisms with different traits.
- **Adaptation**: Organisms can adapt based on environmental changes.
- **Simulation Environment**: Various environments that affect the survival of organisms.

## Getting Started
1. Clone the repository: `git clone https://github.com/i312057-Mao/evolution-game.git`
2. Navigate to the project directory: `cd evolution-game`
3. Install dependencies: `npm install`
4. Start the simulation: `npm start`

## Gameplay Instructions
- **Designing Organisms**: Use the interface to select traits for your organisms.
- **Running Simulations**: Observe how your organisms evolve over time.
- **Analyzing Results**: After running simulations, analyze the survival rates and adaptations of your organisms.
